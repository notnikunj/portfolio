INDIAN SIGN LANGUAGE TRANSLATOR

For my final year project, I developed a real-time Indian Sign Language (ISL) alphabet recognition system using a deep learning approach to bridge communication gaps for the Deaf and hard-of-hearing. The task involved curating a dataset of ~51,000 static ISL gesture images and applying data augmentation. Thenafter, a ResNet50 model was fine-tuned via transfer learning, achieving 96.4% validation accuracy. It faced challenges with visually similar letters like 'J' and 'K'. The model was integrated into a real-time application using OpenCV for webcam input. The model then processes frames to display predicted letters instantly, demonstrating a functional prototype that operates efficiently on standard hardware. This project lays a foundation for future expansion into dynamic gestures and full ISL vocabulary.

VOYAGER APP

Voyager is a mobile travel companion I developed to solve the problem of spontaneous and efficient trip planning. Built with React Native for a cross-platform iOS and Android experience, the app features a clean, modern UI styled with Tailwind CSS and is powered by Google Places API for its location intelligence.

The app's primary functionality allows users to discover points of interest such as landmarks, museums, restaurants, and parks within a specified radius of their current location. By leveraging the Google Places API, Voyager provides reliable data including user photos and description. The core user journey involves exploring these suggestions and then building a personalized day itinerary. Users can add or remove destinations, visualize the order of their trip, and have all essential information consolidated in one place.

DATA CLASSIFICATION FOR DIABETES

This case study explores the use of Principal Component Analysis (PCA) and Support Vector Machine (SVM) classifiers for diabetes diagnosis by analyzing health indicator data from two distinct groups: physically inactive (NoActivity) and active (PhysActivity) individuals. After normalizing the data and using PCA for exploratory analysis, which showed partial but incomplete class separation, group-specific SVM models with an RBF kernel were tuned, achieving test accuracies of 73.4% for the NoActivity group and 74.7% for the PhysActivity group. A critical cross-evaluation revealed limited model generalizability, as the PhysActivity model's accuracy dropped to 65% on the NoActivity test set, demonstrating that population-specific models are necessary due to the significant impact of lifestyle on the underlying patterns in the data.
